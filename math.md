# Решение транспортной задачи для вариантов 1–22

Во всех вариантах рассматривается закрытая (сбалансированная) транспортная задача с тремя пунктами поставки $A_1,A_2,A_3$ и пятью пунктами потребления $B_1,\dots,B_5$. Требуется минимизировать суммарные затраты
$$F=\sum_{i=1}^{3}\sum_{j=1}^{5} c_{ij}x_{ij}$$
при ограничениях баланса по запасам и потребностям, $x_{ij}\ge 0$.

Для каждого варианта последовательно построены: (i) опорный план методом северо-западного угла; (ii) опорный план методом минимальных элементов; (iii) оптимальный план методом потенциалов (MODI).

- Потребности потребителей: $b_1=270$, $b_2=130$, $b_3=190$, $b_4=150$, $b_5=110$.

Матрица удельных затрат (стоимостей) $C=(c_{ij})$:

|---|---|---|---|---|---|
| A1 | 24 | 50 | 55 | 27 | 16 |
| A2 | 50 | 47 | 23 | 17 | 21 |
| A3 | 35 | 59 | 55 | 27 | 41 |

### Опорный план методом северо-западного угла
| | B1 | B2 | B3 | B4 | B5 |
|---|---|---|---|---|---|
| A1 | 200 | 0 | 0 | 0 | 0 |
| A2 | 70 | 130 | 150 | 0 | 0 |
| A3 | 0 | 0 | 40 | 150 | 110 |

**Суммарные транспортные издержки:**  
\(F = 28620\).

### Опорный план методом минимальных элементов (минимальной стоимости)

|---|---|---|---|---|---|
| A1 | 90 | 0 | 0 | 0 | 110 |
| A2 | 0 | 10 | 190 | 150 | 0 |
| A3 | 180 | 120 | 0 | 0 | 0 |

**Суммарные транспортные издержки:**

### Выбор наилучшего опорного плана
